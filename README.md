## Usage

Add `/node_modules/` and `/coverage/` to your `.gitignore`.

## Requirements

- Vue
- Vue-Material
- MySQL
- Express
- Node.js

## Development/Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8081
npm run dev

# run jest test suite
npm run test

# build for production with minification
npm run build
```
